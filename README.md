# FunOverF1

Formalization project for the field with one element in Lean.

The goal is to formalize Deitmar's and To&euml;n-Vaqui&eacute;'s approaches to the
field with one element, following Vezzani's paper
[Deitmar's versus To&euml;n-Vaqui&eacute;'s schemes over F_1](https://rdcu.be/frfIZ),
and ultimately prove that the two approaches are equivalent.
